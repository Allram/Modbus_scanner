Java Modbus Scaner

Project for NetBeans 6.5 with java 1.6
The project uses RXTX libraries (www.rxtx.org).
Tested on Windows XP, Vista 64, WIndows 8/8.1 and Suse Linux 10.0
For Windows the rxtxSerial.dll library is used. Put the dll in system32 folder.
For Linux the librxtxSerialParalkel.so is used. Put the lib in /User/bin folder.
Fo other system download the libraries from www.rxtx.org and try.
